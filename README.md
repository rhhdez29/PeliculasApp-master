# News App - Ionic

Aplicación de películas de la sección 10 del curso de Ionic por Fernando Herrera


## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
npm install
```

## Run app

```bash
ionic serve
```
